# ربات رسمی انجمن علمی مهندسی شیمی دانشگاه محقق اردبیلی

بهترین، کامل‌ترین و حرفه‌ای‌ترین ربات تلگرامی انجمن دانشجویی ایران

## امکانات
- ثبت‌نام هوشمند با تأیید دو مرحله‌ای
- چک عضویت در کانال
- ثبت‌نام رایگان و پولی + رسید (فقط عکس)
- ظرفیت اصلی + ظرفیت کمکی + لیست انتظار
- تأیید/ناخوانا/ابطال رسید توسط ادمین
- گزارش‌گیری کامل + خروجی اکسل
- نظرسنجی + دریافت حداکثر ۴ عکس
- یادآوری رویداد ساعت ۱۸ روز قبل
- هوش مصنوعی Grok-3 (کاملاً فارسی و رایگان)
- بلاک لیست + مدیریت ادمین
- شروع دوباره

  
##توسعه‌دهندگان

مهدی لطفعلی پور (ایده‌پرداز و توسعه دهنده و مدیر پروژه)

ربات با عشق برای دانشجویان مهندسی شیمی دانشگاه محقق اردبیلی ساخته شد
